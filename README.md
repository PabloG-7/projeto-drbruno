# 🏥 Dr. Bruno Ribeiro – Fisioterapeuta

Website profissional desenvolvido para fisioterapeuta, com foco em presença digital, geração de leads e agendamento de pacientes.

## 🚀 Funcionalidades

- Página institucional responsiva (mobile-first)
- Integração com WhatsApp para captação de pacientes
- Estrutura otimizada para SEO básico
- Layout focado em conversão e experiência do usuário

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
